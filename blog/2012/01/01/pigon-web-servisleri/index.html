Pigon Web Servisleri

Jan 1st, 2012

Pigon Web Servisleri, yazılım geliştiricilerinin ihityaç duyabileceği verileri API olarak sunan bir sistemdir.Sistem üzerinde sunulan servislerin kaynak kodu dahil olmak üzere hepsi açık kaynaklıdır. Proje şu anda Name, University ve Location API ile hizmet vermektedir.

Name API : İsimden cinsiyet bilgisini sunan bir API.
University API : Türkiyedeki üniversitelerin listesini fakülte bölüm bilgileri de dahil sunan bir API.
Location API : Türkiyedeki şehir bilgilerini il-ilçe-semt-mahalle şeklinde sunan bir API.

Diğer API ler için çalışmalarım devam etmekte. Eğer projeye katkıda bulunmak isterseniz elinizdeki dataları bana gönderebilirsiniz. Ben de bunların API’lerini yazar sisteme koyarım. Böylece verilere ulaşabileceğimiz ortak bir nokta oluşturabiliriz.
Her türlü öneri, görüş ve eleştirinizi bekliyorum.
